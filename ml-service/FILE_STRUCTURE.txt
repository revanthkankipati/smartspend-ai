DEPLOYMENT OPTIONS

Local Development:
  run.bat (Windows) or ./run.sh (Linux/macOS)

Docker:
  docker build -t smartspend-ml .
  docker run -p 8000:8000 smartspend-ml

Docker Compose:
  docker-compose up ml-service

Production (Gunicorn):
  gunicorn -w 4 -k uvicorn.workers.UvicornWorker app:app
